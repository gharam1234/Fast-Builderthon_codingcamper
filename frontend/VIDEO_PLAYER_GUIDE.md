# 강의 동영상 플레이어 가이드

## 개요

Yeoul 플랫폼의 강의 동영상 플레이어는 고급 기능을 제공하는 현대적인 비디오 재생 환경입니다. 강사의 강의를 효율적으로 학습할 수 있도록 설계되었습니다.

## 주요 기능

### 1. 비디오 재생 제어
- **재생/일시정지**: 플레이 버튼으로 재생 제어
- **진행 바**: 드래그하여 원하는 시점으로 이동
- **시간 표시**: 현재 시간 / 전체 시간 표시
- **음량 조절**: 음량 버튼으로 음소거/해제

### 2. 재생 속도 조절
- **0.5배속**: 천천히 학습할 때
- **1배속**: 기본 속도
- **1.5배속**: 빠른 복습
- **2배속**: 빠른 콘텐츠 스캔

### 3. 화면 제어
- **전체화면**: 최대 화면 크기로 감상
- **설정**: 자막 및 화질 설정
- **마우스 호버**: 마우스가 화면에 올라갈 때만 컨트롤 표시

## 사용 방법

### 동영상 재생하기

1. **강좌 입장**
   - 강의 목록에서 강좌 선택
   - 강의 대시보드의 동영상 플레이어 확인

2. **재생 시작**
   - 큰 재생 버튼 클릭
   - 또는 스페이스바 입력

3. **진행 바 사용**
   - 진행 바 드래그로 원하는 시점 이동
   - 마우스 호버로 시간 미리보기

### 속도 조절

```
[0.5x] [1x] [1.5x] [2x]
```

- 바쁠 때는 **1.5배** 또는 **2배속** 사용
- 어려운 부분은 **0.5배속**으로 천천히 학습

### 음량 제어

- 음량 아이콘 클릭: 음소거/복구
- 음량 슬라이더: 세밀한 조절 (향후 업데이트)

### 화면 제어

- **설정 아이콘**: 화질, 자막 설정 (향후 업데이트)
- **전체화면 버튼**: 최대 크기로 감상

## 학습 통계

### 강의 정보
강의 플레이어 하단에서 다음 정보 확인:

| 항목 | 설명 |
|------|------|
| 재생 시간 | 강의의 전체 길이 |
| 재생 속도 | 현재 선택된 재생 속도 |
| 진행률 | 강의 진행 상황(%) |

### 진행도 추적

- 자동 저장: 강의 진행도는 자동으로 저장됨
- 95% 이상: 강의 완강으로 표시
- 다음 방문: 마지막 시청 위치부터 재개

## 강의 구조

### 챕터 정보

각 강의는 다음 정보를 포함합니다:

- **챕터 제목**: 강의 주제
- **설명**: 강의 내용 요약
- **소요 시간**: 예상 학습 시간
- **학습 목표**: 배우게 될 주요 내용
- **완강 상태**: 강의 수료 여부

### 학습 목표

강의마다 다음과 같은 학습 목표를 제시합니다:

✅ 기본 개념 이해  
✅ 실전 기술 습득  
✅ 프로젝트 적용  
✅ 심화 학습  

## 강의 자료

### 다운로드

- **강의 자료 다운로드**: 강의 관련 문서, 코드 등
- **지원 형식**: PDF, ZIP, CODE 등
- **접근**: 챕터 정보 섹션의 다운로드 버튼

### 공유

- 강의 링크 복사
- SNS 공유
- 동료와 함께 학습

## 팁 및 권장사항

### 효과적인 학습법

1. **영상 시청 전**
   - 학습 목표 확인
   - 필기구 준비

2. **시청 중**
   - 중요한 부분은 일시정지하고 필기
   - 이해 안 되는 부분은 뒤로 돌려서 재청취
   - 질문 작성

3. **시청 후**
   - 요약하기
   - 손글씨 OCR로 필기 정리
   - 퀴즈/과제 풀기
   - 토론 참여

### 학습 속도 가이드

| 상황 | 권장 속도 | 효과 |
|------|---------|------|
| 새로운 내용 | 1.0x ~ 0.75x | 충분한 이해 |
| 복습 | 1.5x ~ 2.0x | 효율적 검토 |
| 어려운 부분 | 0.5x ~ 0.75x | 깊이 있는 학습 |
| 정상 시청 | 1.0x | 강사의 의도 파악 |

### 최적의 학습 환경

- 🔇 조용한 환경
- 💻 큰 화면 (27인치 이상 권장)
- 🪑 편한 의자
- ☀️ 적절한 조명
- ⏱️ 집중할 수 있는 시간

## 기술 사양

### 지원 포맷
- **비디오**: MP4, WebM, HLS
- **음성**: AAC, MP3
- **해상도**: 1080p, 720p, 480p

### 시스템 요구사항
- **브라우저**: Chrome, Safari, Firefox, Edge (최신 버전)
- **인터넷**: 안정적인 고속 인터넷
- **저장소**: 진행도 저장을 위한 LocalStorage

## 자주 묻는 질문

### Q: 동영상을 다운로드할 수 있나요?
**A**: 저작권 보호를 위해 직접 다운로드는 불가능합니다. 강의 자료는 다운로드 가능합니다.

### Q: 재생 기록은 얼마나 유지되나요?
**A**: 브라우저의 LocalStorage에 저장되며, 캐시 삭제 시 초기화됩니다.

### Q: 인터넷이 끊기면 어떻게 되나요?
**A**: 현재는 오프라인 재생을 지원하지 않습니다. (향후 업데이트 예정)

### Q: 여러 기기에서 진행도를 동기화할 수 있나요?
**A**: 현재는 단일 기기만 지원합니다. (계정 동기화 기능은 향후 추가 예정)

### Q: 자막을 켤 수 있나요?
**A**: 향후 업데이트에서 자막 기능이 추가될 예정입니다.

## 문제 해결

### 동영상이 재생되지 않습니다

**확인 사항:**
- 인터넷 연결 확인
- 브라우저 캐시 삭제
- 다른 브라우저 시도
- 브라우저 업데이트

### 재생이 끊깁니다

**해결 방법:**
- 인터넷 속도 확인
- 낮은 화질 선택
- 브라우저 탭 최소화
- 다른 앱 종료

### 진행도가 저장되지 않습니다

**확인 사항:**
- LocalStorage 활성화 확인
- 시크릿 모드 비활성화
- 저장소 여유 확인
- 브라우저 설정 확인
